# Speedrun.com Scraper Benchmark

This is a benchmark test for different kinds of speedrun.com API responses. The aim is to understand which is the best library to parse this data for mass data-collection.

## Context

We will be comparing:

- [bytedance/sonic](https://github.com/bytedance/sonic)
- [json-iterator/go](https://github.com/json-iterator/go)
- [mailru/easyjson](https://github.com/mailru/easyjson)
- [minio/simdjson-go](https://github.com/minio/simdjson-go)
- [buger/jsonparser](https://github.com/buger/jsonparser)

## Comparison Methodology

We will be comparing some data used in the scraping process, such as:
- Games Data
- Leaderboards Data
- Runs Data
- Profile Data
